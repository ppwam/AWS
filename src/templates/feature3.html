如何安全設定 AWS S3 連線

為遵循「不在程式碼中硬編碼金鑰」的安全最佳實踐，請在啟動本系統前，在您的系統環境變數中設定以下金鑰，本程式會自動安全偵測：

Windows PowerShell 設定指令

$env:AWS_ACCESS_KEY_ID="您的_AWS_ACCESS_KEY_ID"
$env:AWS_SECRET_ACCESS_KEY="您的_AWS_SECRET_ACCESS_KEY"
$env:AWS_DEFAULT_REGION="us-east-1"
python run.py

Windows CMD (命令提示字元) 設定指令

set AWS_ACCESS_KEY_ID=您的_AWS_ACCESS_KEY_ID
set AWS_SECRET_ACCESS_KEY=您的_AWS_SECRET_ACCESS_KEY
set AWS_DEFAULT_REGION=us-east-1
python run.py

ℹ️ 您的 AWS 帳號或角色必須擁有對儲存桶 bucket-for-hoodini 的 s3:PutObject、s3:ListBucket 及 s3:DeleteObject 讀寫權限。
